# `turbo_decoder_top` Package

This folder contains the exported architecture package for the **active folded top** [`rtl/turbo_decoder_top.vhd`](../../rtl/turbo_decoder_top.vhd).

This package is intentionally separate from [`../parallel8_top/`](../parallel8_top/):
- `parallel8_top` documents the saved paper-aligned backup
- `turbo_decoder_top` documents the current folded single-SISO implementation
